# 서울 외식업 상권 폐업률 예측 시스템

## 프로젝트 개요
서울시 외식업 상권 데이터를 기반으로 폐업률을 예측하고
창업자에게 최적 상권을 추천하는 머신러닝 시스템

- 분석 기간: 2019 ~ 2024년
- 분석 단위: 서울 25개 자치구 × 7개 업종 × 6개 연도
- 최종 데이터셋: 1,050행 × 29컬럼

## 담당 파트
- 데이터 수집 및 전처리
- 탐색적 데이터 분석 (EDA)

## 주요 발견
- 코로나 기간(2020~2021) 폐업률 억제 후 2022년 이후 급등
- 커피·음료 업종 평균 생존년수 3.9년으로 최단
- 직장인 고정수요 있는 상권(종로·중구)이 폐업률 최저
- 경쟁강도 절대값보다 전년대비 변화율이 더 강한 신호

## 기술 스택
Python, Pandas, NumPy, Matplotlib, Seaborn

## 데이터 출처
서울 열린데이터광장 (data.seoul.go.kr)
- 서울시 일반·휴게음식점 인허가 정보
- 상권분석서비스 (유동인구, 추정매출, 직장인구, 임대료, 영역)
- 지하철 승하차 인원
